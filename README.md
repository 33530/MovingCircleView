#在项目中添加此组件的方式:
##Step 1. 项目的build.gradle文件做如下修改
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
	
```

##Step 2. 添加依赖
```
	dependencies {
	        compile 'com.github.Brioal:MovingCircleView:1.0'
	}
	
```
